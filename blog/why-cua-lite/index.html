Why CUA-Lite — and a call for contributors.

Computer-use agent research is fragmented. Every benchmark ships its own harness, its own action space, its own data format. Every dataset picks its own schema, so data collected for one agent can't train another. Every lab rebuilds the same plumbing — screenshot loops, VM setup, eval scripts — before running a single experiment.

CUA-Lite is one data platform plus one framework, built to democratize this work: one schema for data, one action space for benchmarks, one command for any agent — desktop, web, and mobile. Here is how each kind of contribution compounds:

Datasets

Convert a dataset to LiteSample once and every agent can train on it — today's and tomorrow's. No data of your own? It's all free on Hugging Face.
