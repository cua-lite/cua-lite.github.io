Quickstart

A trajectory in five lines.

Sample a rollout with any computer-use agent. What comes back is a LiteSample — ready to evaluate, SFT, or RL.

quick_start.py
$ uv sync --extra quick-start>>> import lite.gym as gym, lite.agents as agents>>> env = gym.make("lite.demo@create_file", max_steps=10)>>> agent = agents.make("gpt-5.5", env=env)>>> result = await agent.sample(env) # → LiteSample
