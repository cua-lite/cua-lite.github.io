Quick Start

$ uv sync --extra quick-start
>>> import lite.gym as gym, lite.agents as agents
>>> env   = gym.make("lite.demo@create_file")
>>> agent = agents.make("gpt-5.5", env=env)
>>> result = await agent.sample(env)   # -> LiteRLSample
