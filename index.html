The toolkit

One interface, the whole loop.

01

Any agent × any environment

10+ built-in agents — GPT, Claude, Qwen, UI-TARS, MAI-UI — and one unified action + observation space across desktop, web, and mobile. Drop any agent into any env.

env = gym.make("lite.osworld@…")
02

One data schema

Every rollout lands in LiteSample — fine-tune from 10+ preprocessed corpora, or distill any teacher via rollouts.

result.lite_sample # ready to train
03

Eval & RL, one command

Score any agent on any benchmark, or train it with GRPO on highly optimized tasks — CUAGym, WebGym, MobileGym — on top of Slime.

python rollout.py --env-id osworld

Quickstart

A trajectory in five lines.

quick_start.py
$ uv sync --extra quick-start

>>> import lite.gym as gym, lite.agents as agents
>>> env    = gym.make("lite.demo@create_file", max_steps=10)
>>> agent  = agents.make("gpt-5.5", env=env)
>>> result = await agent.sample(env)   # → LiteRLSample
