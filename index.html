[ 01 ] the toolkit

Everything to build computer-use agents

Eval, data, and RL behind one schema and one interface — across every agent and environment.

Any agent

10+ built-in agents — GPT, Claude, Qwen, UI-TARS, MAI-UI — or compose your own from the blocks we ship.

agents.make("gpt-5.5", env=env)

Any environment

One unified action + observation space across Desktop, Web, and Mobile. Drop any agent into any env with near-zero overhead.

gym.make("lite.osworld@...")

Standardized data

One schema — LiteSample — for all data. Fine-tune at scale from Corpora, or on demand from Rollouts.

Eval any benchmark

One command scores any agent on OSWorld, WebArena, WebVoyager, AndroidWorld, MobileWorld, CUABench, and more.

RL any environment

Train any agent with RL on highly optimized tasks — CUAGym, WebGym, MobileGym — GRPO and beyond on Slime.

Lightweight by design

One gym.make / agents.make interface, an env-server for scale, and rollouts that return ready-to-train samples.

[ 03 ] quick start

A trajectory in five lines

Install, pick an agent and an env, sample. That's the whole loop.

quick_start.py
$ uv sync --extra quick-start

>>> import lite.gym as gym, lite.agents as agents
>>> env    = gym.make("lite.demo@create_file", max_steps=10)
>>> agent  = agents.make("gpt-5.5", env=env)
>>> result = await agent.sample(env)   # -> LiteRLSample (reward · steps · data)
