Land it in five lines

Install, pick an agent and an env, sample. That's the whole loop.

quick_start.py
$ uv sync --extra quick-start

>>> import lite.gym as gym, lite.agents as agents
>>> env    = gym.make("lite.demo@create_file", max_steps=10)
>>> agent  = agents.make("gpt-5.5", env=env)
>>> result = await agent.sample(env)   # -> LiteRLSample (reward · steps · data)
